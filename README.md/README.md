# Cancer Survival Prediction using R


## Overview

This project performs survival analysis of lung cancer patients using R.

The analysis includes:

- Kaplan-Meier survival estimation
- Survival comparison between patient groups
- Cox proportional hazards regression
- Clinical risk factor analysis


## Dataset

Dataset:
NCCTG Lung Cancer Data

Contains clinical information from lung cancer patients including:

- survival time
- patient status
- age
- gender
- performance score


## Methods

1. Data preprocessing

2. Kaplan-Meier survival analysis

3. Cox proportional hazards modelling

4. Risk factor interpretation


## Results

Key findings:

- Survival probability decreases over time.
- Patient characteristics influence survival outcomes.
- Clinical variables show different levels of association with mortality risk.


## Tools Used

- R
- survival package
- survminer
- ggplot2
- dplyr
